1) Собрать приложение на эмуляторе.

-загрузить исходный код. Для начала разархивировать папку .zip
-в Android Studio жмём new-open и далее выбираем папку, в которую разархивировали
-всё загрузилось. Там высветится путь другой, нужно подправить на свой путь
-создаём эмулятор
-на открытой странице с исходным кодом нажимаем рядом с нашим созданным эмулятором на кнопку run 'app'
-начнётся сборка приложения под андроид
-по итого приложение откроется на эмуляторе созданном.
-приложение собрано, запущено, можно тестить.

2) Собрать приложение на реальном телефоне

-подсоединить телефон к компьютеру
-войти в режим разработчика
-нажать туглы:
отладка по USB
установка через USB
проверять приложение при установке ADB/ADT
проверять байт-код приложения.

-если всё настроено верно, то заходим в Android Studio и там виден наш подсоединённый телефон.
-напротив нашего телефона нажимаем run 'app'
-начинается сборка приложения на  реальный телефон
-приложение установлено на телефон.

3) Собрать .apk файл и установить на реальном телефоне.
apk (Android Pakage Kit)

-исходный код загружен в Android Studio
-нажимаем Build-Builds Bundle(s)/APK(s)-Build APK(s)
-начинается сборка apk файла
-после сборки высветится сообщение
-жмём в сообщении locate
-нас перенаправляет на место, куда лёг файл .apk
-копируем, отправляем в Telegram
-скачиваем на телефон
-тестируем приложение.
